# Smart Interview Preparation System

An AI-powered interview preparation platform designed to help students and job seekers improve their interview readiness through intelligent resume analysis, skill-gap identification, and personalized learning recommendations.

The system analyzes uploaded resumes, compares them with selected job roles, identifies strengths and areas for improvement, and provides customized interview questions, resume feedback, and learning resources to support effective technical interview preparation.

---

## Key Features

* Resume parsing and analysis
* Skill extraction from resumes
* Job role–based skill comparison
* Skill-gap identification
* Personalized interview question generation
* Resume improvement recommendations
* Learning resource recommendations
* ATS-focused resume evaluation
* Interactive web interface

## Workflow

1. Upload a resume.
2. Select the target job role.
3. Extract skills from the resume.
4. Compare extracted skills with role requirements.
5. Identify strengths and missing skills.
6. Generate personalized interview questions.
7. Recommend relevant learning resources.
8. Provide actionable resume improvement suggestions.
